# Nexus

Nexus é um SaaS de gerenciamento de projetos focado em engenharia de software backend.

## Stack
- Next.js (App Router)
- Prisma + PostgreSQL
- JWT Auth
- TailwindCSS + shadcn/ui
- Arquitetura Service / Repository

## Conceitos demonstrados
- Multi-tenant
- Autorização por role
- Middleware de segurança
- Audit logs
